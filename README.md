####2. 프로젝트 개요 
##### 연구 문제: 최적의 클린로드 시스템 운영을 위한 위치 선정
##### 연구 목표 : 열섬현상이 심각한 지역을 뽑아 클린로드 시스템 설치가 필요한 지역 선정
-> 열섬현상은 미세먼지 농도, 녹지 면적, 생활인구수, 교통량 등의 다양한 요인에 영향을 받음

-> 여러 데이터를 전처리 및 취합해 최적의 위치 선정
